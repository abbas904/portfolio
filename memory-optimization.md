# تحسين أداء السيرفر

## 🚀 حلول مشكلة البطء في npm run dev

### 1. استخدام السكريبتات المحسنة:
```bash
# السيرفر السريع
npm run dev:fast

# السيرفر المحسن بالكامل
npm run dev:optimized

# تنظيف وإعادة تشغيل
npm run dev:clean
```

### 2. تحسينات تم تطبيقها:

#### أ. إعدادات Next.js:
- ✅ تفعيل Turbo Mode
- ✅ تحسين استيرادات الحزم
- ✅ تحسين الذاكرة
- ✅ تحسين Webpack

#### ب. تحسين الذاكرة:
- ✅ زيادة حد الذاكرة إلى 4GB
- ✅ تعطيل Telemetry
- ✅ تحسين Cache

#### ج. تحسين الاستيرادات:
- ✅ تحسين استيرادات Swiper
- ✅ تحسين استيرادات React Icons
- ✅ تحسين استيرادات Radix UI

### 3. نصائح إضافية:

#### أ. تنظيف دوري:
```bash
npm run clean
```

#### ب. إعادة تثبيت الحزم:
```bash
rm -rf node_modules package-lock.json
npm install
```

#### ج. استخدام Yarn (أسرع):
```bash
npm install -g yarn
yarn install
yarn dev
```

### 4. مراقبة الأداء:
- استخدم `npm run dev:fast` للحصول على أفضل أداء
- راقب استخدام الذاكرة في Task Manager
- استخدم `npm run clean` عند مواجهة مشاكل

### 5. إعدادات النظام:
- تأكد من وجود مساحة كافية في القرص الصلب
- أغلق البرامج غير الضرورية
- استخدم SSD للحصول على أداء أفضل
